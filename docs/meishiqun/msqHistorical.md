# 发展历史

3366美食群🍒标准的完整名称是《3366美食大战老鼠群》，旧称《3366美食大战老鼠2群》由于1群荒废多年后来进行统一，去掉了 “2群”后缀，不再在现有行政管理上进行区分，不过1群与2群依旧事实上存在，但更多作为历史研究目的。

3366美食群在《美食大战老鼠》游戏群世界中拥有悠久的历史，我群的前身3366美食大战老鼠官方1群最早于2012年就已创立。

关于3366美食1群，在2013年初由于不可抗力（实为被人举报封禁等原因），最早的3366美食大战老鼠官方群（1旧1群）从此消失。
因此留存到现在的1群，实际上已经是新1群。在旧的1群封禁后，新的1群成立，管理人员也经历了一些变更，但具体的完整变化已经无从可考。
