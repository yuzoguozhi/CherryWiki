# 发展历史

开服专区是《Minecraft》小型服务器游戏。目前已更替了许多周目。服务器最早于2016年就已成立，服务器最初为早期人类间进行Minecraft PE版联机，而后在2017年左右不断进化为基岩版、2018年进化为《Minecraft》JAVA版（PC平台）， 且一直为玩家提供服务器级特色玩法。时至今日，已从早期人类联机到目前运营多款各具特色的服务器游戏。 并于2020年2月公测国际版的《KFZQ 4TH》，且玩家注册量、在线人数、活跃度和赞助收入突破历史。 现如今已推出多款不同玩法的 Minecraft 服务器。

### 紫晶小镇时期
2015年末，在「紫晶家族群」中，平淡无奇的一天，人们得知伟大群主Pockyqiqi宣告她将进入《Minecraft》这个权威的领域内，迎来了包括寂陌等在内的开服元老们一起共同玩过的第一款MC服务器，为后续打下了深厚基础，后来人们将这段时间开启的服务器统称为「紫晶小镇」，为后续开服专区的诞生打下了深厚的基础。

紫晶小镇内人口虽不多，却发生过许多难忘趣事，在以伟大群主Pockyqiqi同志为核心的引领下，包括寂陌、柚子果汁、露恩、寒冰、梦梦等人先后在小镇上崭露头角。

在这个世界刚刚创建时，一切还是原初的混沌。而伟大群主到来后，世界为之焕然一新，然后服初的这一位开始和旧世界的主人们，数位恐怖大王开战。那恐怖的大王们是僵尸、骷髅、熊、蜘蛛…后来，Pockqiqi先召唤了露恩等人，随后是寂陌、柚子果汁等人的到来。据说，他们最初的代号并非与现在相同，有人唤作「潘莎」、有人唤作「xperia」、「弄吃」等，具体其谁已不可知，毕竟——史诗的故事从开天辟地时期总是这样带着些许模糊色彩的。也可能是编者出于隐私性人道保密（）

### 开服起始
2016年，腐竹寂陌再次将深远的战略目标放在了《Minecraft》这款游戏上，做出了影响世界线变化的重要一步也可能是暑假平凡的一日里想与小伙伴们玩游戏了。据说，如今的开服专区起初并不称作这个名字，而是叫做「开房专区」，此后寂陌与小伙伴们开展了多次冒险故事。

「开房专区」最早建立时，使用的还是《Minecraft PE版》，与联机平台「多玩我的世界联机盒子」。

### 网易时期
2018年，随着我的世界中国版推出在即，开房专区也即将与时俱进，推出了基于我的世界中国版的联机周目。此时携带版也已整合升级为基岩版，整体标志着开房专区进入了中国版&基岩版时期。

在2018年的一个夏天，柚子果汁目光坚定着看向了远方，仿佛下了一个重要的决定总之就是再次将深远的战略目标……于是也加入其中，此时开房专区仍叫这个名字，但在柚子果汁加入不久后，由于网易的命名屏蔽规则等原因，只好改个名字……于是在此之后，开房专区便改称为开服专区了。而进入网易时期后，首个周目便称作为——「开服专区Pro」。根据专家朱椿分析，这最初可能是代表着多玩向网易升级性、变革性的一种称呼，也可能只是寂陌随便取的名字。

## 作品简史

* 2016年：早期人类开始进行PE版联机。
* 2018年：《开服专区Pro》上线，成为基岩版历史经典之一。
* 2018年末：《开服专区1st》上线，首次试水PC端 Java版本。
* 2019年初：《开服专区2nd》上线，小试牛刀之作。
* 2019年夏季：《开服专区3rd》上线，短时间内迅速受到玩家喜爱。
* 2020年初：《KFZQ 4th》上线，首次推出国际版服务器，达成里程碑式突破。
* 2021年8月：《KFZQ Skyisland》上线，将2nd中的空岛玩法全新升级回归。
* 2022年3月：《KFZQ Pixelmon》上线，全新尝试使用像素宝可梦玩法，受到广泛欢迎与喜爱。
* 2022年7月：《金土豆群岛》上线，带来 1.7.10 时代玩法的经典复刻。
* 2022年10月：《KFZQ Pro Max》上线，经典 Pro 以更 Max 的极限模式回归。
* 2023年初：《2023新春会》活动服务器上线，再次达到单日在线人数峰值。
* 2023年2月：Linezq品牌宣言发表，开服专区启用双品牌战略，同时推出了首款服务器《Linezq Pack5》，昭示着开服专区冲击高端* 的关键一年。
* 2025年1月26日：《KFZQ 5th：Litetown/莉特镇》正式上线。
